# Random-muisic
🎵 Random Audio Player A Python script that shuffles and plays 16 local MP3 recordings (Enregistrement.mp3) in a random order.  Features:  Randomized Playback: Uses Python's random module.  Direct Integration: Works with files in the root directory.  Linux Ready: Optimized for Ubuntu environments.  Run: python3 main.py

A lightweight Python script that shuffles and plays 16 local MP3 recordings in a randomized sequence.

### 🚀 Features
* **Randomized:** Uses Python `random` for unique playback.
* **Simple:** Plays `Enregistrement.mp3` files directly from the root.
* **Linux Ready:** Optimized for Ubuntu.
